# meep-public-pages

Public static pages for the Meep mobile app, hosted on GitHub Pages.

## Live URLs

- Privacy Policy: <https://trevornogues.github.io/meep-public-pages/privacy.html>
- Landing: <https://trevornogues.github.io/meep-public-pages/>

Use the Privacy Policy URL in App Store Connect.

## Updating the privacy policy

1. Edit `privacy.html`.
2. Bump the "Last updated" date near the top.
3. Commit and push to `main` — Pages redeploys in ~1 minute.

## Enabling GitHub Pages (one-time)

Repo on github.com → **Settings → Pages** → Source: **Deploy from a branch** → Branch: `main` / `(root)` → **Save**.
